How It Works

Interactive guide to your first flag

Step 1

Read the hint

Each level includes a short briefing. It tells you what skill to learn and where to look.

Try it

Open the level page, read the clue, then take the smallest action you can (click, view source, inspect).

Step 2

Find the flag

Flags look like this. They are short text strings that prove you solved the level.

flag{first_steps}
flag{inspect_the_dom}
Step 3

Use the flag

Paste the flag on the home page to unlock the next level. Save it so you can return later.

Why it matters

Flags track progress without accounts. They are your key to move forward.

Bonus

Safe mindset

Practice on this site only. Be respectful so the server stays up for everyone.
