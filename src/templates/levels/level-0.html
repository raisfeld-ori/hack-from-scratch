Level 0

Inside the page

To start becoming a hacker, we first need to learn how to look at the website's internals.

Open the Dev tools.
On Google Chrome - Press F12 or right-click anywhere on the page and select "Inspect".
On Firefox - Press F12 or right-click anywhere on the page and select "Inspect Element".

Use this video for reference: Developer Tools Tutorial

Once you've opened the Dev tools, try searching for the 'flag' word, find it and submit it here.
